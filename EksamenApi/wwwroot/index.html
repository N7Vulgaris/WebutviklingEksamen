The Game Shelf - API

Endpoints:

  • https://localhost:7022/api/videogame
  • https://localhost:7022/api/videogame/{id}
  • https://localhost:7022/api/videogame/GetByTitle/{title}
  • https://localhost:7022/api/videogame/GetByPlatform/{platform}
  • https://localhost:7022/api/videogame/GetByDeveloper/{developer}

How to use the API:

Using the endpoints above, replace the parameter inside of the {} to retrieve the desired video game.

Example: https://localhost:7022/api/videogame/GetByTitle/bloodborne will show these results:

Data available:

When using http requests againts this API, the following information / datatype is utilized:

  • Id: string
  • Title: string
  • Platform: string
  • ReleaseYear: int
  • Image: string
  • Developer: string
  • Publisher: string
  • Genre: string
  • ReviewScore: int
